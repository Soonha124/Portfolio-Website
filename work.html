Mobile App Development

LexaScan

- Text Extractor App (OCR) Have you ever needed to copy text from a photo, scanned document, handwritten note, or screenshot but had to type everything manually? This application was built to solve that everyday problem by making text extraction fast, accurate, and effortless.
The LexaScan is an Android application developed using Kotlin and Jetpack Compose . It allows users to select any image from their device's gallery, automatically detects the text within the image using Optical Character Recognition (OCR) technology, and displays the extracted content in a clean, readable format below the selected image. With a single tap, users can copy the extracted text and use it anywhere—whether it's for notes, assignments, emails, documents, or messaging apps. The application features a modern and intuitive user interface, a custom splash screen, and a smooth user experience designed with simplicity in mind. Throughout the development process, I focused on creating an application that is both practical and user-friendly while gaining hands-on experience in Android app architecture and UI development with Jetpack Compose. This project was developed during my third year of Bachelor's as part of a Mobile App Development Bootcamp organized by Sukkur IBA University. It gave me valuable experience in building real-world Android applications, integrating OCR functionality, designing responsive user interfaces, and delivering a complete application from concept to implementation.
Key Features
  • Extract text from images selected from the device gallery.
  • Fast and accurate Google ML Kit OCR-based text recognition.
  • One-tap copy functionality for quick reuse of extracted text.
  • Clean and responsive user interface built with Jetpack Compose.
  • Custom splash screen for a polished user experience.

Technologies Used
  • Kotlin
  • Jetpack Compose
  • Android Studio
  • Google ML Kit Text Recognition (OCR)
  • Material Design

Taskify

- Smart To-Do & Task Management App Staying organized is one of the biggest challenges in our daily lives. Whether it's managing assignments, office work, personal goals, or important reminders, keeping track of every task can be overwhelming. Taskify was developed to provide a simple, organized, and efficient way for users to manage their daily activities without missing important deadlines.
The Taskify is a modern Android application developed using Dart and Flutter. It enables users to create their own account, securely log in, and manage their personal tasks in an organized environment. Users can create new tasks by providing a title, description, start date, end date, and selecting a category such as Work, Study, Personal, or any other relevant category. Once saved, tasks are grouped according to their selected category, making them easy to browse and manage from the home screen. To help users stay productive, the application sends a notification when a task reaches its scheduled due time, ensuring that important deadlines and reminders are never missed. Users can also access their profile information, manage their account, and securely log out whenever needed.
Throughout the development process, I focused on building a clean, intuitive, and responsive user interface while implementing practical features that solve real-world productivity challenges. This project was developed during my fourth year of Bachelor's as part of a 2-month Mobile App Development Training Program. It strengthened my understanding of Flutter development, state management, user authentication, task organization, notifications, and creating complete mobile applications from planning to deployment.
Key Features
  • User registration, login, and secure authentication.
  • Create tasks with title, description, start date, end date, and category.
  • Organize and browse tasks by categories such as Work, Study, and Personal.
  • Receive reminder notifications when tasks reach their scheduled due time.
  • View profile information and securely log out of the application.

Technologies Used
  • Flutter
  • Dart
  • Android Studio
  • Local Notifications
  • Material Design

BankEase

- Money Transfer Simulation Understanding how money transfers work between bank accounts is a fundamental concept in banking applications. This project was developed to simulate a basic banking system where users can view customer information and perform secure money transfers between predefined accounts.
The Simple Banking App is an Android application developed using Kotlin and Jetpack Compose. It displays a list of predefined customers on the home screen. By selecting a customer, users can view their account details, including their name, email address, and current account balance. The application also provides a money transfer feature that allows users to select a sender and receiver from the customer list and enter the amount to transfer. Once the transaction is completed, the specified amount is automatically deducted from the sender's account and added to the receiver's account, demonstrating the core logic behind fund transfers. This project was developed during my Bachelor's studies as part of my Android development learning journey. It helped me strengthen my understanding of Jetpack Compose, state management, navigation, data handling, and implementing transaction logic within a mobile application.
Key Features
  • View a list of predefined bank customers.
  • Display customer details including name, email, and account balance.
  • Transfer money between customer accounts.
  • Automatically update sender and receiver balances after each transaction.
  • Simple and user-friendly interface built with Jetpack Compose.

Technologies Used
  • Kotlin
  • Jetpack Compose
  • Android Studio
  • SQLite Database
  • Material Design

FinTrack UI

- Personal Finance Management App Managing personal expenses becomes much easier when financial information is presented in a clear and organized way. This project was developed as a personal challenge to improve my Android UI development skills by designing a modern expense tracking application using Jetpack Compose.

The project focuses on creating an intuitive user experience while following modern Material Design principles. Although this is a lightweight application, it helped me gain practical experience in building responsive layouts, designing reusable UI components, and structuring screens commonly found in finance and expense management applications. This project was developed as part of my Android development learning journey and challenged me to recreate a professional-looking financial interface while strengthening my Jetpack Compose skills.
Key Features
  • Modern expense tracking dashboard.
  • Transaction and expense overview.
  • Clean financial summary interface.
  • Responsive UI built entirely with Jetpack Compose.
  • Developed as a UI challenge to improve Android development skills.

Technologies Used
  • Kotlin
  • Jetpack Compose
